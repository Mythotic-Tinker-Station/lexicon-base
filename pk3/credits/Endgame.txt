* Play Information *

Game                    : DOOM2
Map #                   : 1 - 8
Single Player           : Yes
Cooperative 2-4 Player  : Yes
Deathmatch 2-4 Player   : Yes on some maps. Try maps 1, 2, 4. Map 8 might play ok too.
Other game styles       : COMPET-N style modes such as speed, tyson, etc.
Difficulty Settings     : Yes

* Some maps (especially map 6) has levels of detail which may cause 
HOM in doom2 exe. The HOM's are not widespread and only viewed from 
certain angles so they should not effect doom2 exe gameplay.
